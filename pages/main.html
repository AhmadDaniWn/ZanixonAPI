What is zanixon API?


Zanixon API is a free rest API that can be used by everyone without usage limits, Zanixon API was made as my learning material about javascript and express.js and this rest API is the second version of the previous one. The difference is that the previous one has a few endpoints and uses the easy-api.ts package but finally I felt something was missing so I decided to use express.js and make the second version of the Zanixon API, For the first version of the rest API I have turned off because the server is used for this second API.
